## projet6OC

API du projet 6 de OpenClassrooms.
Retrouvez les dépendances dans le fichier package.json.

## démarrer le serveur

Pour faire fonctionner le serveur, utilisez la commande `npm server` qui utilisera le port `3000`.
Retrouvez le front-end sur le repo GitHub de OpenClassroom : `https://github.com/OpenClassrooms-Student-Center/dwj-projet6`
